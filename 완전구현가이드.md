# Implementation Guide

This guide is intentionally short for the public OSS candidate.

## Local Setup

```powershell
python -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
python -m pip install -r requirements.txt pytest
Copy-Item .env.example .env
python run.py
```

## Validation

```powershell
python -m compileall -q .
python -m pytest -q
python scripts/check_oss_readiness.py --repo-root .
```

## Provider Credentials

Use environment variables only. Do not write real provider credentials into
source files, Markdown files, test fixtures, or issue reports.

## Release Readiness

The local target is `PASS: Auto-Verified`. Public GitHub issue creation, release
creation, and OpenAI form submission remain `NOT_PERFORMED` until the maintainer
executes them.
