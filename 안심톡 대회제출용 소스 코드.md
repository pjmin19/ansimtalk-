# Archived Contest Source Snapshot

This public OSS candidate no longer stores a full source-code snapshot in this
Markdown file. The runnable source of truth is the repository code itself.

Reason:

- Old contest snapshots can drift away from the application code.
- They can duplicate configuration examples that look like credentials.
- They make public maintenance and review harder.

Use these files instead:

- `README.md` for setup and project scope.
- `SECURITY.md` for credential handling.
- `docs/ROADMAP.md` for DTT milestones.
- `scripts/check_oss_readiness.py` for local readiness validation.
