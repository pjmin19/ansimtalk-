# Code of Conduct

This project expects respectful, practical collaboration.

Participants should:

- Keep discussions focused on the work.
- Avoid harassment, threats, insults, and personal attacks.
- Handle safety-sensitive examples without exposing private people or students.
- Accept maintainer decisions about scope, security, and release readiness.

Reports can be sent to the maintainer through the repository contact channel.
The maintainer may remove comments, issues, or pull requests that make the
project unsafe or hostile.
