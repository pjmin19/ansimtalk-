# Operating Principle Summary

AnsimTalk separates evidence collection from judgment.

## Evidence Collection

- Store upload metadata.
- Calculate a SHA-256 hash.
- Keep generated report files out of git.

## Optional Analysis

External providers can be connected through environment variables. When
credentials are missing, tests should exercise fallback behavior instead of
calling live services.

## Human Review

AI-assisted output is only a draft signal. A human must review the report before
real-world action.

## OSS Readiness

Run the readiness validator before publishing a branch or release:

```powershell
python scripts/check_oss_readiness.py --repo-root .
```
